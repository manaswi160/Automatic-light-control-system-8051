Automatic Light Control System Using LDR and Relay with 8051

Project Overview
This project implements an Automatic Light Control System using an LDR (Light Dependent Resistor), Relay, and 8051 Microcontroller. The system automatically turns ON the light in low-light conditions and turns it OFF when sufficient light is available.

Components Used
8051 Microcontroller
LDR Sensor
Relay Module
Bulb/LED
Resistors
Power Supply
Connecting Wires

Working Principle
The LDR senses ambient light intensity. The 8051 microcontroller processes the sensor input and controls the relay accordingly. When darkness is detected, the relay switches ON the light; when brightness increases, the relay switches OFF the light.

Applications
Automatic Street Lighting
Home Automation
Garden Lighting
Energy Saving Systems
